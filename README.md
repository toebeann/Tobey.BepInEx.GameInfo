# Tobey.BepInEx.GameInfo

A simple BepInEx plugin which logs information about the current Unity game.

## Usage

Just plop the contents of the downloaded .zip from [the releases page](https://github.com/toebeann/Tobey.BepInEx.GameInfo/releases) into your game folder (after installing [BepInEx](https://github.com/BepInEx/BepInEx), of course).
